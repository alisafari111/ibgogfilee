# 📁 دانلودر گوگل درایو با اینترنت ملی 
با استفاده از این ریپازیتوری شما میتوانید فایل های درون گوگل درایو را به درون گیتهاب خود منتقل کرده و آنها را به راحتی دانلود کنید. این پروژه برای واکشی لیست تمام فایل های درون پوشه های گوگل درایور و همچنین دانلود سالم همه ی آنها طراحی شده است.
هر لینک گوگل درایور دو حالت دارد : ۱ - تک فایل ۲- فولدری که درون آن پوشه ها و فایل های گوناگون است.

- برای حالت اول مستقیم فایل دانلود میشود.
- برای حالت دوم لیست تمام فایل ها به همراه لینک دانلود استخراج میشود و سپس بصورت دسته دسته دانلود و درون ریپازیتوری شما قرار میگیرند تا تمام فایلهای درون لیست تمام شود.

## ⚙️ آموزش نصب و استفاده

- وارد اکانت گیتهاب خود شوید.
- از بالای همین صفحه روی گزینه ی fork بزنید.
- در صفحه جدید نام فورک را انتخاب و سپس create fork را بزنید.
- در فورک ساخته شده در گیتهاب خودتان به سربرگ Actions بروید.
- سمت چپ قسمت All workflows امکانات در دسترس میباشد.
- اگر لینک شما فولدر است روی گزینه ی ((  لیست کردن فایلها  )) بزنید.
- در سمت راست روی run workflow بزنید.
- در کادر باز شده آدرس گوگل درایو را وارد کنید و دکمه run workflow را بزنید.
- لیست تمام فایلهای درون فولدر در فایلی به نام list.txt ساخته میشود.
- اگر فایل ها بالای ۶۰ عدد است باید آنها را ۶۰ تا ۶۰ تا دانلود کنید.
- تمام مراحل خودکار انجام میشود روی ورک فلو به نام (( دانلود از گوگل درایو  )) بزنید و آنرا اجرا کنید
- این ورک فلو دنبال فایل list.txt میگردد و تمام لینک های درون آن را دانلود و به فایلی به نام done.txt منتقل میکند.
- این فرآیند باید ادامه پیدا کند تا تمام فایلهای در انتظار درون list.txt به داخل done.txt منتقل شود.
- فایل ها درون یک پوشه هم نام با پوشه ی گوگل درایو برای شما قرار گرفته است که میتوانید آنها را دانلود کنید.
